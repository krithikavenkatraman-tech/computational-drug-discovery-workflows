# Binding Pocket Prediction

## Overview

Binding pocket prediction identifies potential ligand-binding sites on a protein surface before molecular docking. Computational tools analyze the protein structure to detect cavities, pockets, and surface grooves that are likely to accommodate small molecules.

This module demonstrates binding site prediction using PrankWeb and CB-Dock2.

---

## Objectives

- Predict ligand binding pockets
- Compare pocket prediction methods
- Select an active site for molecular docking
- Visualize predicted binding cavities

---

## Workflow

1. Upload protein structure
2. Predict binding pockets
3. Rank predicted pockets
4. Select the most probable active site
5. Use the predicted pocket for docking

---

## Software Used

- PrankWeb
- CB-Dock2
- Discovery Studio Visualizer

---

## Learning Outcome

This module demonstrates computational identification of protein binding sites prior to molecular docking.
