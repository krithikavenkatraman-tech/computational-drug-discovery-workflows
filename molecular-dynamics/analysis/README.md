# Analysis

After completion of the Molecular Dynamics simulation, trajectory files are analyzed to evaluate the structural stability and dynamic behavior of the biomolecular system.

The analyses included in this module are:

- RMSD
- RMSF
- Radius of Gyration
- SASA
- Hydrogen Bond Analysis

Each analysis provides different information regarding the stability, flexibility, compactness, solvent exposure, and intermolecular interactions of the simulated protein.
