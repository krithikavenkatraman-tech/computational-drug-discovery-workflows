# ADMET Prediction

## Overview

ADMET (Absorption, Distribution, Metabolism, Excretion, and Toxicity) prediction evaluates the pharmacokinetic and toxicity properties of compounds during the early stages of drug discovery.

Computational ADMET analysis helps prioritize compounds with favorable drug-like properties before experimental validation.

---

## Objectives

- Predict pharmacokinetic properties
- Evaluate drug-likeness
- Assess toxicity risks
- Identify promising lead compounds

---

## Workflow

1. Obtain compound structure
2. Upload to ADMET prediction server
3. Predict pharmacokinetic properties
4. Evaluate drug-likeness
5. Assess toxicity
6. Interpret results

---

## Software Used

- SwissADME
- pkCSM

---

## Learning Outcome

This module demonstrates computational evaluation of pharmacokinetic and toxicity properties to support lead optimization in drug discovery.
