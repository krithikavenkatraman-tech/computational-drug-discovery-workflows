# Molecular Docking

> Structure-based molecular docking workflow performed using AutoDock4 for protein–ligand interaction analysis.

---

# Overview

Molecular docking is a computational technique used to predict the preferred binding orientation of a ligand within the active site of a target protein. It plays a key role in structure-based drug discovery by estimating binding affinity and identifying important molecular interactions.

This repository documents the molecular docking workflow performed using **AutoDock4**, including protein preparation, ligand preparation, docking configuration, and docking analysis.

---

# Objectives

- Prepare protein and ligand structures for docking
- Configure docking parameters using AutoDock4
- Predict protein–ligand binding poses
- Evaluate binding affinity
- Analyze molecular interactions
