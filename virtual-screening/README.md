# Virtual Screening using PyRx

## Overview

Virtual screening is a computational technique used to rapidly evaluate a library of compounds against a target protein to identify potential binders. PyRx integrates AutoDock Vina and Open Babel to simplify ligand screening and docking workflows.

---

## Objectives

- Prepare ligand libraries
- Screen multiple compounds
- Rank compounds by binding affinity
- Select promising lead molecules

---

## Workflow

1. Import protein structure
2. Import ligand library
3. Energy minimization
4. Configure docking parameters
5. Perform virtual screening
6. Rank compounds
7. Select top hits

---

## Software Used

- PyRx
- AutoDock Vina
- Discovery Studio Visualizer

---

## Learning Outcome

This module demonstrates the use of PyRx for high-throughput virtual screening and identification of potential lead compounds.
