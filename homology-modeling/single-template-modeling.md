# Single Template Homology Modeling

## Objective

Generate a three-dimensional protein structure using a single experimentally resolved template.

---

## Target Protein

RIPK2

---

## Template

PDB ID: 6ES0

---

## Workflow

1. Retrieve protein sequence
2. Identify suitable template
3. Perform sequence alignment
4. Generate structural models using MODELLER
5. Evaluate model quality
6. Select the best model

---

## Model Evaluation

Model quality was assessed using:

- DOPE Score
- GA341 Score
- Structural inspection

---

## Applications

- Molecular docking
- Binding site prediction
- Structural analysis
